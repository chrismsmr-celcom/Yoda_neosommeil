# 🧘 Yoda Travel Agent - DeepSeek

Assistant de voyage vocal utilisant l'IA DeepSeek pour l'extraction intelligente des données de voyage.

## ✨ Fonctionnalités

- 🎤 **Reconnaissance vocale** : Parlez naturellement en français
- 🤖 **IA DeepSeek** : Analyse intelligente du texte
- 📊 **Extraction structurée** : Récupère origine, destination, date, durée
- 🎯 **Function Calling** : Utilisation des outils DeepSeek
- 🎨 **Interface Yoda** : Design inspiré de Star Wars
